# smartcar2023
## 第十八届全国大学生智能汽车竞赛百度智慧交通创意组备赛项目
🤣1.HSV_test.py:HSV阈值分割调试程序
🤣2.rename.py:批量重命名程序
🤣3.follow.py:基于分割线递归拟合中线的智能车巡线方案调试程序
🤣4.newfollow.py:基于赛道追踪拟合中线的智能车巡线方案调试程序
🤣5.midfollow.py:基于赛道追踪拟合中线（常规）和基于中线扫描与补线（环岛）的智能车巡线方案调试程序
